## Android Tech Stack</h2>

- Minumum SDK 23
- Kotlin
    - Coroutines & Flow 
- Dagger-Hilt
- CameraX
- Android Jetpack

    - Navigation Component
    - DataBinding & ViewBinding
- Firebase
    - Firebase Analytics

    - Room
    - Gson
- Remote

    - OkHttp3
- Glide
- Material Components
- ML Kit
- Timber
